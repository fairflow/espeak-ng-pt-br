# Phrase List Format

## Overview
The Portuguese Pronunciation Practice app supports importing phrase lists for structured, sequential practice sessions.

## File Format
- **File type**: Plain text (.txt)
- **Encoding**: UTF-8
- **Format**: One phrase per line
- Empty lines are automatically ignored

## Practice Modes

### Free Text Mode (Default)
- Type any word or phrase you want to practice
- No restrictions on content or order
- Best for: spontaneous practice, testing specific words

### Guided List Mode
- Activated when you upload a phrase list file
- Shows current phrase prominently
- Navigate with Previous/Next buttons
- Jump to specific phrase with number input
- Progress indicator shows your position (e.g., "Phrase 5 of 20")
- All practice features remain available (audio playback, recording, analysis)

## Navigation Features

1. **Previous/Next Buttons**: Move sequentially through the list
2. **Jump to Phrase**: Enter a phrase number (1-based) and click "Go"
3. **Progress Bar**: Visual indicator of completion
4. **Current Phrase Display**: Large, clear display of the phrase to practice

## Tips for Creating Phrase Lists

### Beginner Lists
- Start with common greetings and courtesy phrases
- Use short phrases (2-5 words)
- Group by theme (greetings, questions, responses)

### Intermediate Lists
- Include varied sentence structures
- Mix formal and informal language
- Add phrases with challenging sounds (nasal vowels, diphthongs)

### Advanced Lists
- Longer sentences (8-12 words)
- Idiomatic expressions
- Minimal pairs for fine-tuning pronunciation

## Best Practices

1. **Focus on Quality**: 10-20 well-chosen phrases are better than 100 random ones
2. **Practice Consistently**: Complete the same list multiple times to track improvement
3. **Theme Your Lists**: Group related phrases for contextual learning
4. **Progressive Difficulty**: Start each session with easier phrases, end with challenging ones
5. **Review Mode**: Create a list of phrases you previously scored low on
